O Que é Amazon QuickSight? Seu Estúdio de BI na Nuvem! ☁️🎨

  • Serverless BI: Sem instalar ou gerenciar servidores — a AWS cuida de tudo.
  • Escalável: Atende desde poucos a milhares de usuários com dashboards rápidos.
  • Integrado: Conecta-se nativamente ao S3, Athena, Glue, Redshift, RDS e mais.
  • Dashboards Interativos: Filtros, drill-down, visuais dinâmicos.
  • Pagamento por Sessão: Custo-efetivo para muitos usuários esporádicos.
  • Insigths de ML: Machine Learning embutido para detectar padrões automaticamente.

As Partes do Painel de Controle Mágico 🖼️📊

  • Data Source: Origem dos dados (Glue/Athena/S3, Redshift, RDS, uploads).
  • Dataset: Seleção, preparação, joins e campos calculados. Pode usar SPICE.
  • SPICE: Motor in-memory super rápido para dashboards ágeis.
  • Analysis: Workspace visual para criar gráficos (drag-and-drop).
  • Visualization: Gráfico ou tabela individual do painel.
  • Dashboard: Coleção final de visuais e controles, pronta para compartilhar.
  • Stories: Tour guiado e narrativo pelo dashboard.

Como Montar Seu Painel Mágico (O Fluxo no QuickSight) 🛠️🎨📊

  1. Conectar aos Dados: Novo dataset, escolha a fonte (ex: Athena via Data Catalog).
  2. Preparar Dataset: Escolher tabelas, fazer joins, criar campos calculados, decidir usar SPICE ou query direta.
  3. Criar Análise: Arrastar campos, escolher tipo de gráfico, customizar visualizações.
  4. Adicionar Interatividade: Filtros, parâmetros, drill-downs.
  5. Publicar Dashboard: Salvar e publicar seu painel.
  6. Compartilhar: Gerencie o acesso, compartilhe o link com quem precisa.

Conectando ao Tesouro (S3 via Athena/Catalog) 🔗📦🗣️

  • QuickSight conecta facilmente a tabelas catalogadas (Data Catalog) que apontam para dados processados no S3.
  • Otimizações do S3 (Parquet, particionamento) ajudam a performance dos dashboards.
